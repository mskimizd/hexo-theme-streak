# hexo-theme-streak
A Theme for Hexo

[Live Demo](https://mskimizd.github.io/)

## Usage

### Install Plugin

```bash
$ npm install --save hexo-renderer-less
```

### Download theme

```bash
$ git clone -b master https://github.com/???.git themes/streak
```

### Configure

In file `hexo/_config.yml`, change the theme to `streak`

## Enjoy :smile:
